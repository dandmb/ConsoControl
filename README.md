<h1 align="center">📦 RappelConso KMP (Under development)</h1>

<p align="center">
Application mobile Kotlin Multiplatform permettant de consulter les rappels produits de l’API officielle RappelConso.
</p>

  <img src="https://img.shields.io/badge/Kotlin-Multiplatform-7F52FF?style=for-the-badge&logo=kotlin&logoColor=white"/>

</p>

---

<h2>✨ Fonctionnalités</h2>

<ul>
  <li>📦 Consultation des rappels produits en temps réel</li>
  <li>🔎 Recherche par mot-clé</li>
  <li>🏷️ Filtrage par catégorie</li>
  <li>📱 Support Android & iOS</li>
  <li>🌙 Mode sombre</li>
  <li>🔄 Synchronisation avec l’API officielle</li>
  <li>🧩 Architecture propre et modulaire</li>
  <li>📷 Scanner le code barre d'un produit</li>
</ul>

---

<h2>🛠️ Stack technique</h2>

<ul>
  <li><strong>Kotlin Multiplatform</strong></li>
  <li><strong>Compose Multiplatform</strong></li>
  <li><strong>Ktor</strong></li>
  <li><strong>Kotlinx Serialization</strong></li>
  <li><strong>Coroutines & Flow</strong></li>
  <li><strong>MVVM / Clean Architecture</strong></li>
  <li><strong>Room DB</strong></li>
  <li><strong>Koin DI</strong></li>
</ul>
